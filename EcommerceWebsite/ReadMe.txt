--> To run the project you need to configure the firebase project and it's api keys to run the setup

--> In Backend you need to add the key.js file for the firebase API keys for backend to work

--> In frontend you need to add config.js file for the firebase API keys for frontend to work

--> Need to hit the command 'npm run dev' to run both the sides

--> Make sure to run the backend before frontend for API calls from frontend to work.
